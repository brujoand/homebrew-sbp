# SBP Homebrew Tap

## How do I install SBP?
`brew install brujoand/sbp/sbp`

Or `brew tap brujoand/sbp` and then `brew install sbp`.

## Documentation
`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).

## Update SBP with Homebrew

```bash
brew update && brew upgrade
```

## Uninstall SBP with Homebrew

```bash
brew uninstall brujoand/sbp/sbp
```
